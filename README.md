# Jsearch

Jsearch a simple script that grep infos from javascript files

## Getting Started

pip3 install --user -r requirements.txt
python3 jsearch.py -u https://foo.bar

### Prerequisites

python3.0 >= 3.7
pip

### Features
All listed strings will be dump inside domain javascript files

* AWS endpoints
* AWS Keys
* All URLS
* Upload fields
